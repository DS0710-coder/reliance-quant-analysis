# Reliance Industries - Quant Analysis

A moving average crossover strategy built on 2 years of 
Reliance Industries (NSE) stock data.

## What this project covers
- Downloaded real stock data using yfinance
- Calculated daily returns
- Built MA20/MA50 crossover strategy
- Backtested and measured performance
- Sharpe Ratio, Max Drawdown analysis

## Results
- Strategy Return: 6.37% annually
- Beat Buy & Hold (1.55% annually)
- Max Drawdown: -18.57%

## Libraries used
pandas, numpy, matplotlib, yfinance, mplfinance
